# MMWCalc 📐

**Mathematics in the Modern World Calculator**  
A fully offline Android app for GEC 3 (MMW) students.

Built with ❤️ by **Reyny Mark Mabeza** — soloDev

---

## Features

- **18+ topic calculators** covering all MMW topics
- **Smart solve-for engine** — leave one field blank, app solves for it
- **Deep step-by-step solutions** — shows the actual math process
- **Learn section** — formula reference, worked examples, tips & tricks
- **Calculation history** — grouped by date, swipe to delete, re-open with pre-filled values
- **Normal distribution bell curve** visual
- **Modular arithmetic** with 5 operations
- **Finance calculators** — Simple/Compound Interest, Stocks, Bonds, Mutual Funds, Loans, Credit Cards
- **100% offline** — no internet required

---

## Topics Covered

### Sequences
- Fibonacci Sequence (constraint solver with custom seeds)
- Arithmetic Sequence
- Geometric Sequence

### Statistics
- Central Tendency (Ungrouped & Grouped)
- Measures of Dispersion (Ungrouped & Grouped)
- Measures of Relative Position (Ungrouped & Grouped)
- Normal Distribution
- Linear Regression
- Correlation (Pearson's r)

### Number Theory
- Modular Arithmetic (basic, addition, subtraction, multiplication, division)
- Zeller's Congruence

### Finance
- Simple Interest
- Compound Interest
- Stocks
- Bonds
- Mutual Funds
- Loans (Amortization)
- Credit Cards

3. Let Gradle sync
4. Run on device or emulator (API 26+)
